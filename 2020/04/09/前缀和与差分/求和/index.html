求和
发表于 2020-04-09| 更新于 2020-04-10|算法前缀和与差分
阅读量:

要求的是：$\sum\limits_{i=1}^n(x_i+z_i)(ID_x+ID_z)$ （注：$x_i$是纸条编号为$ID_x$时候的值）

我们将这个式子化简得到：$\sum\limits_{i=1}^n(x_iID_x+x_iID_z+z_iID_x+z_iID_z)$ （*)

再观察这个题目，发现可行的一对$(ID_x, ID_z)$ 必须满足的条件是：$ID_x$和$ID_z$的奇偶性必须相同，这样才能使得点y到点x和点z的距离相等，这个用数学方法很容易证明，如：

（1）若$ID_y$为奇数：

奇数-奇数=偶数，奇数+奇数=偶数； 奇数-偶数=奇数，奇数+奇数=奇数；

（2）同理可证当$ID_y$为偶数的情况

有了上面的结论，就可以把纸带上的点进行分类，首先根据点的奇偶性分出来两个集合，然后把每个集合内部的点根据是否是同一种颜色再分成一个个小集合（储存的是点的$ID$，即奇偶性和颜色都相同的点的集合，使用变长数组$vector$进行储存），储存时注意要按照$ID$的大小从小到大储存，因为x是要小于z的。

当我们对小集合内部进行(*)式的累加时，对其使用数学手段进行优化：

假设这个小集合内部有5个点，分别以$x,y,a,b,z$ 代替，

可以看到每个点都可以与其他的点相连，每个点可以连4条边，但是重复的边要去掉，那么对于这个式子:$\sum\limits_{i=1}^n(x_iID_x+x_iID_z+z_iID_x+z_iID_z)$ （*)

先要理解$(x_iID_x+x_iID_z+z_iID_x+z_iID_z)$表示的是点$x_i$和点$z_i$连成一条边时候的一个三元组的和。那么图中的这5个点可以连成的边有：$(x,y),(x,a),(x,b),(x,z),(y,a),(y,b),(y,z),(a,b),(a,z),(b,z)$ ，每个点都连有4条边，所以(*)式展开后系数为$x_i,y_i,a_i,b_i,z_i $的项各有8个，这样下面的式子化简就好理解了。

只看系数为$xi$的项，考虑到点x和其他所有点都连出一条边，则只包含系数$x_i$的项的总和为：$4xID_x+x(ID_y+ID_a+ID_b+ID_z) = 3xID_x+x(ID_x+ID_y+ID_a+ID_b+ID_z)=3xID_x+x\sum\limits{i=x}^zID_i$

同理可得只包含系数$yi$的项的总和为：$3yID_y+y\sum\limits{i=x}^zID_i$

所以$\sum\limits{i=1}^n(x_iID_x+x_iID_z+z_iID_x+z_iID_z)=\sum\limits{m=x}^z3mIDm+\sum\limits{m=x}^z(m\sum\limits_{n=x}^zID_n)$

推广到n个点，$(n-1)$ 条边，则有：$\sum\limits{i=1}^n(x_iID_x+x_iID_z+z_iID_x+z_iID_z)=\sum\limits{m=x}^z(n-2)mIDm+\sum\limits{m=x}^z(m\sum\limits_{n=x}^zID_n)$

另外值得一提的是，由于$x$与$ID_x$的地位相同，上式还可以写成：

$\sum\limits{i=1}^n(x_iID_x+x_iID_z+z_iID_x+z_iID_z)=\sum\limits{m=x}^z(n-2)mIDm+\sum\limits{m=x}^z(IDm\sum\limits{n=x}^zn)$

此外，题目数据较大，结果要模10007，但为了以防数据范围溢出，我们可以在每一个计算的地方都模10007。

附，模运算公式：

  • (a + b) % p = (a%p + b%p) %p
  • (a - b) % p = ((a%p - b%p) + p) %p
  • (a b) % p = (a%p)(b%p) %p

则有：$\sum\limits{m=x}^z(n-2)mID_m+\sum\limits{m=x}^zIDm\sum\limits{n=x}^zn=\sum\limits{m=x}^z(n-2)\%10007m\%10007ID_m\%10007+\sum\limits{m=x}^z(IDm\%10007\sum\limits{n=x}^zn\%10007)$

根据上面的式子，就容易理解代码了。
